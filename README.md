# Something Ventured

Contracts and frontend for Something Ventured

Stack:

- Hardhat
- Ethers.js
- TypeScript
- NextJS
- ChakraUI
- ESLint
- Prettier

### Running locally

The following tasks are available:

Frontend
```shell
yarn dev
yarn start
yarn build
```

HardHat
```shell
yarn lint
yarn compile
yarn test
yarn chain
yarn deploy
```
